Counting Sort

Counting Sort is a non-comparison-based sorting algorithm that works particularly efficiently when the range of input values is small compared to the number of elements to be sorted. It counts the occurrences of each element and uses this information to place elements in their correct sorted positions.

How Counting Sort Works

  • Find the maximum element in the input array to determine the range.
  • Create a count array of size max+1 and initialize all elements to 0.
  • Count frequency of each element in the input array by incrementing the corresponding index in the count array.
  • Calculate the cumulative sum (prefix sum) for each position in the count array.
  • Build the output array by placing each element from the input array at its correct sorted position using the count array.
Time Complexity Space Complexity
Best Case: O(n+k)
Average Case: O(n+k)
Worst Case: O(n+k)

where n is the size of input array and k is the range of input values

 O(n+k)

where n is the size of output array and k is the range of input values

Advantages

  • Performs faster than most comparison-based sorting algorithms when the range of input values is small.
  • Time complexity is linear O(n+k) when k is not significantly larger than n.
  • It is a stable sorting algorithm - maintains the relative order of equal elements.
  • Simple to implement and understand.

Disadvantages

  • Not efficient for large ranges of values or when the range is much larger than the number of elements.
  • Doesn't work with decimal values directly without modification.
  • Not an in-place sorting algorithm - requires additional space proportional to the range of values.
  • Not suitable for sorting strings or complex objects without modification.

Applications

  • Used for sorting items with a limited range, such as sorting students by grades, ages, or test scores.
  • As a subroutine in Radix Sort to sort numbers with multiple digits.
  • Useful in scenarios where stability of sort is important (preserving original order of equal elements).
  • Efficient for sorting large datasets with small range, like sorting people by age.
  • Used in Bucket Sort to divide elements into different buckets.

Implementation

Below are implementations of the Counting Sort algorithm in different programming languages. Each implementation follows the same basic steps: find the maximum element, create and populate the count array, calculate the prefix sum, and build the sorted output array.

#include <iostream>
#include <vector>
using namespace std;

vector<int> countingSort(vector<int>& arr) {
    int n = arr.size();
    
    // Find the maximum element
    int maxval = 0;
    for (int i = 0; i < n; i++) {
        maxval = max(maxval, arr[i]);
    }
    
    // Create and initialize count array
    vector<int> count(maxval + 1, 0);
    
    // Count frequency of each element
    for (int i = 0; i < n; i++) {
        count[arr[i]]++;
    }
    
    // Compute prefix sum
    for (int i = 1; i <= maxval; i++) {
        count[i] += count[i - 1];
    }
    
    vector<int> ans(n);
    for (int i = n - 1; i >= 0; i--) {
        ans[count[arr[i]] - 1] = arr[i];
        count[arr[i]]--;
    }
    
    return ans;
}

int main() {
    vector<int> arr = {5, 2, 9, 5, 2, 3, 5};
    vector<int> sorted = countingSort(arr);
    
    cout << "Sorted array: ";
    for (int x : sorted) {
        cout << x << " ";
    }
    
    return 0;
}
import java.util.Arrays;

public class CountingSort {
    public static int[] countingSort(int[] arr) {
        int n = arr.length;
        
        // Find the maximum element
        int max = Arrays.stream(arr).max().getAsInt();
        
        // Create and initialize count array
        int[] count = new int[max + 1];
        
        // Count frequency of each element
        for (int i = 0; i < n; i++) {
            count[arr[i]]++;
        }
        
        // Compute prefix sum
        for (int i = 1; i <= max; i++) {
            count[i] += count[i - 1];
        }
        
        // Build the output array
        int[] output = new int[n];
        for (int i = n - 1; i >= 0; i--) {
            output[count[arr[i]] - 1] = arr[i];
            count[arr[i]]--;
        }
        
        return output;
    }
    
    public static void main(String[] args) {

        int[] sorted = countingSort(arr);
        
        System.out.print("Sorted array: ");
        for (int x : sorted) {
            System.out.print(x + " ");
        }
    }
}
def counting_sort(arr):
    # Find the maximum element
    max_val = max(arr)
    
    # Create and initialize count array
    count = [0] * (max_val + 1)
    
    # Count frequency of each element
    for num in arr:
        count[num] += 1
    
    # Compute prefix sum
    for i in range(1, max_val + 1):
        count[i] += count[i - 1]
    
    # Build the output array
    output = [0] * len(arr)
    for i in range(len(arr) - 1, -1, -1):
        output[count[arr[i]] - 1] = arr[i]
        count[arr[i]] -= 1
    
    return output

<?php
function countingSort($arr) {
    $n = count($arr);
    
    // Find the maximum element
    $max = max($arr);
    
    // Create and initialize count array
    $count = array_fill(0, $max + 1, 0);
    
    // Count frequency of each element
    foreach ($arr as $value) {
        $count[$value]++;
    }
    
    // Compute prefix sum
    for ($i = 1; $i <= $max; $i++) {
        $count[$i] += $count[$i - 1];
    }
    
    // Build the output array
    $output = array_fill(0, $n, 0);
    for ($i = $n - 1; $i >= 0; $i--) {
        $output[$count[$arr[$i]] - 1] = $arr[$i];
        $count[$arr[$i]]--;
    }
    
    return $output;
}
