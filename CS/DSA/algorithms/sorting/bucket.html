Bucket Sort

Bucket Sort is a sorting technique that involves dividing elements into various groups, or buckets. These buckets are formed by uniformly distributing the elements. Once the elements are divided into buckets, they can be sorted using any other sorting algorithm. Finally, the sorted elements are gathered together in an ordered fashion.

How Bucket Sort Works

  • Create Buckets: Create n empty buckets (or lists) where n is typically the length of the input array.
  • Distribute Elements: Insert each element of the array into a bucket based on its value. This is typically done by a scatter function that determines which bucket an element should go into.
  • Sort Buckets: Sort individual buckets using another sorting algorithm (commonly insertion sort, which works well for small lists).
  • Gather Elements: Concatenate all sorted buckets to get the final sorted array.
Time Complexity Space Complexity
Best Case: O(n + k)
Average Case: O(n + k)
Worst Case: O(n²)

where n is the number of elements and k is the number of buckets. The worst case occurs when all elements are placed in a single bucket.

Advantages

  • Efficient for uniformly distributed data, achieving linear time complexity O(n) in the best case.
  • Can be more efficient than comparison-based sorting algorithms like Quick Sort or Merge Sort for certain datasets.
  • Works well with floating-point numbers that are uniformly distributed across a range.
  • Easily parallelizable, as each bucket can be sorted independently.

Disadvantages

  • Not an in-place sorting algorithm; requires additional storage for the buckets.
  • Performance heavily depends on the uniform distribution of elements; inefficient when elements are clustered.
  • Can degrade to O(n²) in the worst case when most elements are placed in a single bucket.
  • Requires knowledge about the input range to create appropriate buckets.

Applications

  • Used when input is uniformly distributed over a range, such as sorting a large set of floating point numbers between 0 and 1.
  • Efficient for external sorting where data doesn't fit in memory and must be processed in chunks.
  • Used in computer graphics for sorting polygons by depth (z-buffer algorithm).
  • Histogram creation and analysis in image processing.
  • Used as a subroutine in Radix Sort to sort numbers with multiple digits.

Implementation

Below are implementations of the Bucket Sort algorithm in different programming languages. Each implementation follows the same basic steps: create buckets, distribute elements, sort individual buckets, and gather all elements back into a single sorted array.

#include <iostream>

// Function to sort arr[] of size n using bucket sort
void bucketSort(float arr[], int n) {
    // Create n empty buckets
    vector b[n];
    
    // Put array elements in different buckets
    for (int i = 0; i < n; i++) {
        // Index in bucket
        int bi = n * arr[i];
        b[bi].push_back(arr[i]);
    }
    
    // Sort individual buckets using insertion sort
    for (int i = 0; i < n; i++) {
        sort(b[i].begin(), b[i].end());
    }
    
    // Concatenate all buckets into arr[]
    int index = 0;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < b[i].size(); j++) {
            arr[index++] = b[i][j];
        }
    }
}

import java.util.*;

class BucketSort {
    // Function to sort arr[] of size n using bucket sort
    static void bucketSort(float arr[], int n) {
        if (n <= 0)
            return;
            
        // Create n empty buckets
        @SuppressWarnings("unchecked")
        List[] buckets = new ArrayList[n];
        
        for (int i = 0; i < n; i++) {
            buckets[i] = new ArrayList();
        }
        
        // Put array elements in different buckets
        for (int i = 0; i < n; i++) {
            int bucketIndex = (int) (n * arr[i]);
            buckets[bucketIndex].add(arr[i]);
        }
        
        // Sort individual buckets
        for (int i = 0; i < n; i++) {
            Collections.sort(buckets[i]);
        }
        
        // Concatenate all buckets into arr[]
        int index = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < buckets[i].size(); j++) {
                arr[index++] = buckets[i].get(j);
            }
        }
    }
    
    // Driver program
    public static void main(String args[]) {
        float arr[] = { 0.897f, 0.565f, 0.656f, 0.1234f, 0.665f, 0.3434f };

        bucketSort(arr, n);
        
        System.out.println("Sorted array is ");

            System.out.print(el + " ");
        }
    }
}

function bucketSort(arr, n = arr.length) {
    if (n <= 0) return arr;
    
    // Create n empty buckets
    let buckets = new Array(n);
    for (let i = 0; i < n; i++) {
        buckets[i] = [];
    }
    
    // Put array elements in different buckets
    for (let i = 0; i < n; i++) {
        let bucketIndex = Math.floor(n * arr[i]);
        buckets[bucketIndex].push(arr[i]);
    }
    
    // Sort individual buckets
    for (let i = 0; i < n; i++) {
        buckets[i].sort((a, b) => a - b);
    }
    
    // Concatenate all buckets into arr
    let index = 0;
    for (let i = 0; i < n; i++) {
        for (let j = 0; j < buckets[i].length; j++) {
            arr[index++] = buckets[i][j];
        }
    }
    
    return arr;
}

using System;

using System.Linq;

class BucketSort {
    // Function to sort arr[] of size n using bucket sort
    static void bucketSort(float[] arr, int n) {
        if (n <= 0)
            return;
            
        // Create n empty buckets
        List[] buckets = new List[n];
        
        for (int i = 0; i < n; i++) {
            buckets[i] = new List();
        }
        
        // Put array elements in different buckets
        for (int i = 0; i < n; i++) {
            int bucketIndex = (int)(n * arr[i]);
            buckets[bucketIndex].Add(arr[i]);
        }
        
        // Sort individual buckets
        for (int i = 0; i < n; i++) {
            buckets[i].Sort();
        }
        
        // Concatenate all buckets into arr[]
        int index = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < buckets[i].Count; j++) {
                arr[index++] = buckets[i][j];
            }
        }
    }
    
    // Driver code
    static void Main() {
        float[] arr = { 0.897f, 0.565f, 0.656f, 0.1234f, 0.665f, 0.3434f };
        int n = arr.Length;
        
        bucketSort(arr, n);
        
        Console.WriteLine("Sorted array is:");
        foreach (float item in arr) {
            Console.Write(item + " ");
        }
    }
}
<?php
function bucketSort($arr) {
    $n = count($arr);
    if ($n <= 0) {
        return $arr;
    }
    
    // Create n empty buckets
    $buckets = array_fill(0, $n, []);
    
    // Put array elements in different buckets
    for ($i = 0; $i < $n; $i++) {
        $bucketIndex = floor($n * $arr[$i]);
        $buckets[$bucketIndex][] = $arr[$i];
    }
    
    // Sort individual buckets
    for ($i = 0; $i < $n; $i++) {
        sort($buckets[$i]);
    }
    
    // Concatenate all buckets into arr[]
    $index = 0;
    for ($i = 0; $i < $n; $i++) {
        foreach ($buckets[$i] as $value) {
            $arr[$index++] = $value;
        }
    }
    
    return $arr;
}
