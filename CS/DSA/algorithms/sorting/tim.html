Tim Sort

TimSort is a hybrid sorting algorithm derived from merge sort and insertion sort. It was designed to perform well on many kinds of real-world data. TimSort is the default sorting algorithm used by Python's sort() and sorted() functions, as well as Java's Arrays.sort() for non-primitive types and many other modern programming languages.

The algorithm is designed to exploit the existing order in the data to minimize the number of comparisons and swaps.

How TimSort Works

  • Divide the Array: TimSort first divides the input array into small subarrays called "runs".
  • Sort Runs: Each run is sorted using insertion sort, which performs efficiently on small or partially sorted arrays.
  • Merge Runs: The sorted runs are then merged using a modified merge sort algorithm that includes optimizations to improve performance.
  • Adaptive Algorithm: TimSort adapts its behavior based on the characteristics of the input data, which makes it efficient for a wide range of inputs.

Key Features of TimSort

  • Run Detection: TimSort identifies and exploits natural runs (sequences that are already sorted) in the input data.
  • Galloping Mode: When merging runs, TimSort can switch to galloping mode to skip over large portions of already-sorted data.
  • Minimum Run Size: TimSort ensures runs are at least a minimum size to improve efficiency (typically 32 or 64 elements).
  • Balanced Merges: TimSort uses a stack discipline and merge criteria to ensure balanced merges and maintain O(n log n) worst-case performance.

Advantages

  • Performs exceptionally well on real-world data with many types of patterns.
  • Exploits the natural order that may exist in real-world data.
  • Stable sorting algorithm that preserves the relative order of equal elements.
  • Adaptive algorithm that performs better on partially sorted arrays.
  • Requires fewer comparisons than most other O(n log n) algorithms.

Disadvantages

  • More complex implementation compared to simpler algorithms like quicksort or merge sort.
  • Requires extra memory space for merging operations (not an in-place algorithm).
  • For very small arrays, the overhead might make it less efficient than simpler algorithms.
  • Tuning parameters like minimum run size may need adjustment for optimal performance in different contexts.

Applications

  • Default sorting algorithm in many programming languages including Python and Java (for non-primitive types).
  • Excellent for sorting real-world data that may have partially ordered sequences.
  • Used in database systems for efficient sorting operations.
  • Effective for sorting large datasets where performance is critical.
  • Suitable for applications where stability in sorting is important (preserving relative order of equal elements).

// This function sorts array from left to right using
// insertion sort algorithm
void insertionSort(int arr[], int left, int right) {
    for (int i = left + 1; i <= right; i++) {
        int temp = arr[i];
        int j = i - 1;
        while (j >= left && arr[j] > temp) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = temp;
    }
}

// Merge function to merge the sorted runs
void merge(int arr[], int l, int m, int r) {
    // Original array is broken in two parts
    // left and right array
    int len1 = m - l + 1;
    int len2 = r - m;
    int left[len1], right[len2];
    
    for (int i = 0; i < len1; i++)
        left[i] = arr[l + i];
    for (int i = 0; i < len2; i++)
        right[i] = arr[m + 1 + i];
    
    int i = 0;
    int j = 0;
    int k = l;
    
    // After comparing, we merge those two array
    // in larger sub array
    while (i < len1 && j < len2) {
        if (left[i] <= right[j]) {
            arr[k] = left[i];
            i++;
        } else {
            arr[k] = right[j];
            j++;
        }
        k++;
    }
    
    // Copy remaining elements of left, if any
    while (i < len1) {
        arr[k] = left[i];
        k++;
        i++;
    }
    
    // Copy remaining elements of right, if any
    while (j < len2) {
        arr[k] = right[j];
        k++;
        j++;
    }
}

// Iterative Timsort function to sort arr[0...n-1]
void timSort(int arr[], int n) {
    const int RUN = 32; // can be adjusted based on input size
    
    // Sort individual subarrays of size RUN
    for (int i = 0; i < n; i += RUN)
        insertionSort(arr, i, min((i + RUN - 1), (n - 1)));
    
    // Start merging from size RUN (or 32).
    // It will merge
    // to form size 64, then 128, 256 and so on....
    for (int size = RUN; size < n; size = 2 * size) {
        // Pick starting point of left sub array. We
        // are going to merge arr[left..left+size-1]
        // and arr[left+size, left+2*size-1]
        // After every merge, we increase left by 2*size
        for (int left = 0; left < n; left += 2 * size) {
            // Find ending point of left sub array
            // mid+1 is starting point of right sub array
            int mid = left + size - 1;
            int right = min((left + 2 * size - 1), (n - 1));
            
            // Merge sub array arr[left.....mid] &
            // arr[mid+1....right]
            if (mid < right)
                merge(arr, left, mid, right);
        }
    }
}

                arr[k] = left[i];
                i++;
            } else {
                arr[k] = right[j];
                j++;
            }
            k++;

        while (i < len1) {

        while (j < len2) {

            k++;
            j++;
        }

    // Iterative Timsort function to sort arr[0...n-1]
    static void timSort(int[] arr, int n) {
        // Sort individual subarrays of size RUN
        for (int i = 0; i < n; i += MIN_MERGE) {
            insertionSort(arr, i, Math.min((i + MIN_MERGE - 1), (n - 1)));
        }
        
        // Start merging from size RUN (or 32). It will merge
        // to form size 64, then 128, 256 and so on....
        for (int size = MIN_MERGE; size < n; size = 2 * size) {
            // Pick starting point of left sub array. We
            // are going to merge arr[left..left+size-1]
            // and arr[left+size, left+2*size-1]
            // After every merge, we increase left by 2*size
            for (int left = 0; left < n; left += 2 * size) {
                // Find ending point of left sub array
                // mid+1 is starting point of right sub array
                int mid = left + size - 1;
                int right = Math.min((left + 2 * size - 1), (n - 1));
                
                // Merge sub array arr[left.....mid] &
                // arr[mid+1....right]
                if (mid < right)
                    merge(arr, left, mid, right);
            }
        }
    }

        for (int i = 0; i < n; i++) {

def insertion_sort(arr, left=0, right=None):
    if right is None:
        right = len(arr) - 1

    for i in range(left + 1, right + 1):
        temp = arr[i]
        j = i - 1
        while j >= left and arr[j] > temp:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = temp

def merge(arr, left, mid, right):
    len1, len2 = mid - left + 1, right - mid
    left_arr, right_arr = [], []
    
    for i in range(0, len1):
        left_arr.append(arr[left + i])
    for i in range(0, len2):
        right_arr.append(arr[mid + 1 + i])
    
    i, j, k = 0, 0, left
    
    while i < len1 and j < len2:
        if left_arr[i] <= right_arr[j]:
            arr[k] = left_arr[i]
            i += 1
        else:
            arr[k] = right_arr[j]
            j += 1
        k += 1
    
    while i < len1:
        arr[k] = left_arr[i]
        i += 1
        k += 1
    
    while j < len2:
        arr[k] = right_arr[j]
        j += 1
        k += 1

def tim_sort(arr):
    n = len(arr)
    min_run = 32  # Minimum size of a run
    
    # Sort individual subarrays of size min_run
    for start in range(0, n, min_run):
        end = min(start + min_run - 1, n - 1)
        insertion_sort(arr, start, end)
    
    # Start merging from size min_run
    size = min_run
    while size < n:
        for left in range(0, n, 2 * size):
            mid = min(n - 1, left + size - 1)
            right = min(left + 2 * size - 1, n - 1)
            if mid < right:
                merge(arr, left, mid, right)
        size = 2 * size

// Iterative Timsort function to sort arr[0...n-1]
function timSort(arr) {
    const n = arr.length;
    const MIN_RUN = 32;
    
    // Sort individual subarrays of size MIN_RUN
    for (let i = 0; i < n; i += MIN_RUN) {
        insertionSort(arr, i, Math.min(i + MIN_RUN - 1, n - 1));
    }
    
    // Start merging from size MIN_RUN
    for (let size = MIN_RUN; size < n; size = 2 * size) {
        for (let left = 0; left < n; left += 2 * size) {
            const mid = left + size - 1;
            const right = Math.min(left + 2 * size - 1, n - 1);
            
            if (mid < right) {
                merge(arr, left, mid, right);
            }
        }
    }
    
    return arr;
}
