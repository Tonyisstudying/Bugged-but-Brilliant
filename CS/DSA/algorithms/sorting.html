Sorting Algorithms

Sorting algorithms are used to arrange the elements of a list in a specific order, such as numerical or alphabetical. It organizes the items in a systematic way, making it easier to search for and access specific elements.

Comparison Based Sorting Algorithms

Bubble Sort

Compares and swaps adjacent items repeatedly. Easy to understand but slow for large datasets.

Time Complexity: O(n²) in worst and average cases

Space Complexity: O(1)

Learn More

Selection Sort

Finds the minimum element and puts it at the beginning. Simple but inefficient for large lists.

Time Complexity: O(n²) in all cases

Space Complexity: O(1)

Learn More

Insertion Sort

Builds the sorted list one item at a time. Efficient for small data sets and nearly sorted data.

Time Complexity: O(n²) worst case, O(n) best case

Space Complexity: O(1)

Learn More

Merge Sort

Divides the list into halves, sorts them, and merges. Reliable and efficient.

Time Complexity: O(n log n) in all cases

Space Complexity: O(n)

Learn More

Quick Sort

Picks a "pivot," splits the list around it, and sorts recursively. Fast for most cases.

Time Complexity: O(n log n) average, O(n²) worst case

Space Complexity: O(log n) average

Learn More

Heap Sort

Uses a binary heap data structure to sort elements. Efficient and in-place algorithm.

Time Complexity: O(n log n) in all cases

Space Complexity: O(1)

Learn More

Non-Comparison Based Sorting Algorithms

Counting Sort

Counts occurrences of each element and reconstructs the sorted array. Good for small range of integers.

Time Complexity: O(n+k) where k is the range of input

Space Complexity: O(n+k)

Learn More

Radix Sort

Sorts numbers digit by digit from least to most significant digit. Works well for fixed-length integers.

Time Complexity: O(d*(n+b)) where d is digits and b is base

Space Complexity: O(n+b)

Learn More

Bucket Sort

Distributes elements into buckets and sorts each bucket. Works well for uniformly distributed data.

Time Complexity: O(n+k) average case

Space Complexity: O(n+k)

Learn More

Hybrid Sorting Algorithms

Tim Sort

A hybrid of merge sort and insertion sort. Used in Python and Java standard libraries.

Time Complexity: O(n log n) worst case

Space Complexity: O(n)

Learn More

Intro Sort

Begins with quicksort and switches to heapsort when recursion depth exceeds a certain level.

Time Complexity: O(n log n) worst case

Space Complexity: O(log n)
