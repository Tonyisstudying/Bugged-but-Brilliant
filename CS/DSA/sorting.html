Sorting Algorithms

Sorting algorithms are used to arrange the elements of a list in a specific order, such as numerical or alphabetical. It organizes the items in a systematic way, making it easier to search for and access specific elements.

Bubble Sort

Compares and swaps adjacent items repeatedly. Easy to understand but slow for large datasets.

function bubbleSort(arr) { const n = arr.length; for (let i = 0; i < n; i++) { for (let j = 0; j < n - i - 1; j++) { // Compare adjacent elements if (arr[j] > arr[j + 1]) { // Swap them if they are in wrong order [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]]; } } } return arr; }

Time Complexity: O(n²) in worst and average cases.

Space Complexity: O(1) - Sorts in-place.

When to use: Educational purposes or very small datasets.

Merge Sort

Divides the list into halves, sorts them, and merges. Reliable and efficient.

function mergeSort(arr) { // Base case if (arr.length <= 1) return arr; // Divide array in half const mid = Math.floor(arr.length / 2); const left = mergeSort(arr.slice(0, mid)); const right = mergeSort(arr.slice(mid)); // Merge sorted halves return merge(left, right); } function merge(left, right) { let result = []; let i = 0; let j = 0; // Compare elements from both arrays and add smaller one to result while (i < left.length && j < right.length) { if (left[i] < right[j]) { result.push(left[i]); i++; } else { result.push(right[j]); j++; } } // Add remaining elements return [...result, ...left.slice(i), ...right.slice(j)]; }

Time Complexity: O(n log n) in all cases.

Space Complexity: O(n) - Requires additional space.

When to use: General purpose sorting when stability is needed.

Quick Sort

Picks a "pivot," splits the list around it, and sorts recursively. Fast for most cases.

function quickSort(arr) { // Base case if (arr.length <= 1) return arr; // Select pivot (first element for simplicity) const pivot = arr[0]; // Partition array const less = arr.slice(1).filter(x => x <= pivot); const greater = arr.slice(1).filter(x => x > pivot); // Recursive calls and combine results return [...quickSort(less), pivot, ...quickSort(greater)]; }

Time Complexity: O(n log n) average, O(n²) worst case.

Space Complexity: O(log n) average for call stack.

When to use: General purpose sorting when memory usage is a concern.
